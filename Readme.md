#### This repo was created for IT infrastructure course at TalTech 2020

Reopsitory includes infrastructure as code for a mocked startup

Technologies used: 

- Nginx
- Bind9
- MySQL
- Prometheus
- InfluxDB+Telegraf
- Grafana
- Docker
- HAProxy
- Keepalived
- Duplicity
- Jinja templating
- Bash scripts and cronjobs

Infrastructure is deployed by running `ansible-playbook infra.yaml`


You will also find here grafana dashboard screenshots, infrastructure scheme, backup service level agreement and restore guidlines as well as high availability MySQL setup script as it was not part of the final playbook for final exam.
